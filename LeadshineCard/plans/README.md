## 🏗️ 架构亮点

### 1. 分层架构

```
应用层 → 服务层 → 核心抽象层 → 实现层 → 硬件层
```

每一层都有明确的职责，层与层之间通过接口通信，实现了高内聚低耦合。

### 2. 依赖注入

使用 Microsoft.Extensions.DependencyInjection 实现依赖注入：

```csharp
services.AddLeadshineMotionControl(configuration);
```

一行代码完成所有服务注册，简单优雅。

### 3. 接口抽象

核心接口设计：
- `IMotionCard` - 板卡管理
- `IAxisController` - 轴控制
- `IIoController` - IO控制
- `IInterpolationController` - 插补控制
- `IEncoderController` - 编码器
- `IHomeController` - 回零

### 4. 日志系统

集成 Microsoft.Extensions.Logging，支持：
- 控制台日志
- 文件日志
- 结构化日志
- 日志级别过滤

### 5. 配置管理

使用 appsettings.json 管理配置：
- 板卡参数
- 轴参数
- 日志配置
- 支持热更新
